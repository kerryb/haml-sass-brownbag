What is HAML?

  • Alternative to ERB etc
  • No inline code in HTML
  • Structured

So what does it look like?

            %html
              %head
                %title Hello World
              %body
                %p This is Haml!
          
            <html>
              <head>
                <title>Hello World</title>
              </head>
              <body>
                <p>This is Haml!</p>
              </body>
            </html>
