What is Haml?

  • Alternative to ERB etc
  • Separates code from HTML
  • Structured
  • Used by Hashrocket, among others
  • http://haml-lang.com/

What does it look like?

            !!!
            %html
              %head
                %title Hello World
              %body
                %p This is Haml!

➧

            <!DOCTYPE html>
            <html>
              <head>
                <title>Hello World</title>
              </head>
              <body>
                <p>This is Haml!</p>
              </body>
            </html>

Advantages

  • Guarantees valid HTML
  • Encourages good practices
  • DRY
  • Ensures clean indentation

Disadvantages

  • Something else to learn
  • Harder to create view from HTML
  • Less flexible

Using in Rails

In your GemFile:

        gem "haml"

Views named something.html.haml will be automatically compiled and rendered.

Using outside Rails

        gem install haml

        engine = Haml::Engine.new("%p Haml code!")
        engine.render #=> "<p>Haml code!</p>\n"

Saas

Saas : CSS :: Haml : HTML

http://sass-lang.com/

Formats

.sass

  • Pre v3
  • Similar format to Haml
  • Compiled to CSS

.scss

  • v3 and above
  • Inspired by LESS
  • Superset of CSS

Formats

.sass

            table.hl
              margin: 2em 0
              td.ln
                text-align: right

            li
              font:
                family: serif
                weight: bold
                size: 1.2em

.scss

            table.hl {
              margin: 2em 0;
              td.ln {
                text-align: right;
              }
            }

            li {
              font: {
                family: serif;
                weight: bold;
                size: 1.2em;
              }
            }

Using in Rails

In your GemFile:

        gem "haml"

Stylesheets in public/stylesheets/sass will be compiled and cached as CSS when necessary.

Using outside Rails

        gem install haml

        sass input.scss output.css

        sass --watch app/sass:public/stylesheets

Mixins

            @mixin table-base {
              th {
                text-align: center;
                font-weight: bold;
              }
              td, th {padding: 2px}
            }

            @mixin left($dist) {
              float: left;
              margin-left: $dist;
            }

            #data {
              @include left(10px);
              @include table-base;
            }
          
            #data {
              float: left;
              margin-left: 10px;
            }
            #data th {
              text-align: center;
              font-weight: bold;
            }
            #data td, #data th {
              padding: 2px;
            }

Selector Inheritance

            .error {
              border: 1px #f00;
              background: #fdd;
            }
            .error.intrusion {
              font-size: 1.3em;
              font-weight: bold;
            }

            .badError {
              @extend .error;
              border-width: 3px;
            }
          
            .error, .badError {
              border: 1px #f00;
              background: #fdd;
            }

            .error.intrusion,
            .badError.intrusion {
              font-size: 1.3em;
              font-weight: bold;
            }

            .badError {
              border-width: 3px;
            }
